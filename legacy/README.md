# gnu.in-OS — legacy archive

Frozen provenance of the original separate specs (**Gnosis** ⊕ **Context.Spec** ⊕ **GNU.IN**) that were merged into the gnu.in-OS boards. Official name is **gnu.in-OS** (the `-shell` branding and the hero wallpaper were retired during F0).

- `MANIFEST.json` — sources, canonical paths, dedupe record, findings.
- `SOURCES.sha256` — UTF-8 fingerprints of the key source files (integrity).
- `snapshots/` — superseded design boards kept for the record (Unified System decision board, Index copy).

Live data — tokens.json, motion atoms/molecules, blob.in engine — migrates **into** the system. This folder is memory only.
